### 1 项目背景
作为医药制造及 CDMO 龙头企业，普洛药业面临国内外复杂多变的竞争环境。竞争对手（如合全、齐鲁、Divis 等）的战略动向、产能布局、财务健康度及技术研发进度直接影响我司的决策。

**项目阶段规划：**
- **第一阶段（当前）：** 聚焦核心功能——智能情报分析引擎。采用手工录入数据方式，快速验证八大维度分析模型的有效性
- **第二阶段（后续）：** 逐步接入自动化数据采集，提升数据获取效率

**本期目标：** 先解决"数据处理难、分析维度不系统"的痛点，建立标准化的竞争情报分析框架

### 2 建设目标
构建一套**智能化、可配置化**的竞争情报分析系统，实现：

**本期目标（第一阶段）：**
1.  **灵活配置：** 支持用户自定义添加/编辑竞争对手名单
2.  **智能分析：** 基于 AI 大模型对录入的文档进行八大维度深度分析
3.  **结构化输出：** 将非结构化文档转化为标准化情报报告
4.  **决策支持：** 支持单企业深度分析和多企业对比分析

**远期目标（第二阶段）：**
- 逐步接入自动化数据采集（公告、专利、研报等）

### 3 系统架构（两阶段实施）

#### 3.1 第一阶段：手工录入 + 智能分析（当前）

**数据输入方式：**
| 数据类型 | 输入方式 | 说明 |
| :--- | :--- | :--- |
| 竞争对手名单 | 手工配置 | 用户通过界面添加/编辑竞争对手名称、股票代码、官网等基础信息 |
| 公告/年报 | 文件上传 | 支持PDF、Word、TXT格式批量上传 |
| 研报/新闻 | 文本粘贴 | 支持直接粘贴文本或批量导入 |
| 财务数据 | Excel导入 | 标准化模板导入关键财务指标 |

**核心优势：**
- ✅ 无需等待数据采集开发，立即投入使用
- ✅ 数据质量可控，人工筛选确保输入质量
- ✅ 快速验证八大维度分析模型的有效性
- ✅ 用户可灵活配置分析对象

#### 3.2 第二阶段：自动化数据采集（后续）
待第一阶段功能稳定后，逐步接入：
- 巨潮资讯网公告自动下载（已完成）
- 专利数据API接入
- 研报数据接入
- 官网信息抓取

### 4 智能情报分析引擎（核心）
基于**RAG（检索增强生成）+ 行业大模型**技术，严格按照八大框架进行自动化分析。

| 分析维度 | 自动化处理逻辑 | 输出成果示例 |
| :--- | :--- | :--- |
| **1) 战略分析** | 提取 CEO 致辞、年报“经营讨论”章节，进行语义分析。 | 战略地图、SWOT 矩阵图、风险热力图 |
| **2) 商业模式** | 三级分析深度可选：基础版填充9宫格；进阶版识别商业模式类型+盈利模式；深度版增加竞争对手对比+演变分析。识别创新点/差异化，评估可持续性和风险。 | 9宫格表格+可视化画布、商业模式对比图、风险评估表 |
| **3) 资源能力** | 统计专利数、产能公告、招聘 JD（分析人力方向）、认证资质。 | 资源清单、核心能力雷达图 |
| **4) 财务分析** | 自动抓取三张表，计算比率（ROE, 毛利率，负债率等）。 | 财务趋势图、同业对比表、异常波动预警 |
| **5) 前景分析** | 结合研报、管线进度、扩产计划，预测未来增长点。 | 增长驱动力报告、管线进度条 |
| **6) 发展历程** | 按时间轴提取里程碑事件（融资、并购、获批）。 | 企业发展时间轴 |
| **7) 股权/子公司** | 解析股权穿透图，识别核心利润贡献子公司。 | 股权穿透图、核心子公司列表 |
| **8) 企业文化** | 分析官网价值观、员工评价、社会责任报告。 | 文化关键词云、ESG 评分 |

### 4 情报可视化与报告中心
*   **竞争驾驶舱：** 一屏览尽核心对手关键指标（股价、产能、舆情）。
*   **自动周报/月报：** 系统每周自动生成《竞争情报简报》，推送至管理层邮箱。
*   **深度对比工具：** 支持"普洛药业 vs 合全药业"一键生成对比分析报告。

### 附录A：八大维度详细设计

#### A.1 战略分析维度

**功能定位：** 从企业年报、公告中提取战略信息，进行多维度、多层次分析

**分析模式：**
| 模式 | 适用场景 | 输出内容 |
| :--- | :--- | :--- |
| **浅层分析** | 快速了解、初步筛选 | 战略目标清单、关键举措列表、战略关键词 |
| **中层分析** | 常规研究、定期跟踪 | SWOT矩阵、战略意图解读、核心竞争力识别 |
| **深层分析** | 深度研究、投资决策 | 战略地图、竞争定位图、风险热力图、战略演变趋势 |

**对比分析类型：**
1. **横向对比：** 2家及以上企业同期战略对比
   - 战略定位差异
   - 资源配置对比
   - 竞争策略差异
   
2. **纵向对比：** 单企业3-5年战略演变
   - 战略调整轨迹
   - 关键转折点识别
   - 战略一致性评估

**Prompt模板示例（中层分析）：**
```
【角色】
你是一位资深的医药行业战略分析师，擅长从企业年报中提取和分析战略信息。

【任务】
请对以下{公司名称}年报内容进行中层深度的战略分析。

【分析维度】
1. SWOT分析（优势、劣势、机会、威胁）
2. 战略意图解读（使命、愿景、战略目标）
3. 关键战略举措（未来1-3年重点行动）
4. 核心竞争力识别

【文档内容】
{年报文本片段}

【输出格式】
## 战略分析摘要
[200字以内的战略概况]

## SWOT分析矩阵
| 维度 | 内容 |
| :--- | :--- |
| 优势(Strengths) | ... |
| 劣势(Weaknesses) | ... |
| 机会(Opportunities) | ... |
| 威胁(Threats) | ... |

## 战略意图
- 使命：...
- 愿景：...
- 战略目标：...

## 关键举措
1. ...
2. ...
3. ...

## 核心竞争力
- ...

【输出要求】
- 使用Markdown格式
- 定量数据用表格呈现
- 识别3-5个关键战略信号
- 中文输出
```

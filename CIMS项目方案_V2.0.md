
### 1 项目背景
作为医药制造及 CDMO 龙头企业，普洛药业面临国内外复杂多变的竞争环境。竞争对手（如合全、齐鲁、Divis 等）的战略动向、产能布局、财务健康度及技术研发进度直接影响我司的决策。

**项目阶段规划：**
- **第一阶段（当前）：** 聚焦核心功能——智能情报分析引擎。采用手工录入数据方式，快速验证八大维度分析模型的有效性
- **第二阶段（后续）：** 逐步接入自动化数据采集，提升数据获取效率

**本期目标：** 先解决"数据处理难、分析维度不系统"的痛点，建立标准化的竞争情报分析框架

### 2 建设目标
构建一套**智能化、可配置化**的竞争情报分析系统，实现：

**本期目标（第一阶段）：**
1.  **灵活配置：** 支持用户自定义添加/编辑竞争对手名单
2.  **智能分析：** 基于 AI 大模型对录入的文档进行八大维度深度分析
3.  **结构化输出：** 将非结构化文档转化为标准化情报报告
4.  **决策支持：** 支持单企业深度分析和多企业对比分析

**远期目标（第二阶段）：**
- 逐步接入自动化数据采集（公告、专利、研报等）

### 4 智能情报分析引擎（核心）
基于**RAG（检索增强生成）+ 行业大模型**技术，严格按照八大框架进行自动化分析。

| 分析维度 | 处理逻辑（第一阶段-手工录入+AI分析） | 输出成果示例 |
| :--- | :--- | :--- |
| **1) 战略分析** | 三级分析深度可选：浅层提取战略目标清单；中层SWOT分析+战略意图解读；深层战略地图绘制+竞争定位+风险预警。支持横向对比（多企业）和纵向对比（3-5年历史演变）。 | 战略清单/SWOT矩阵/战略地图、风险热力图、对比分析表 |
| **2) 商业模式** | 三级分析深度可选：基础版填充9宫格；进阶版识别商业模式类型+盈利模式；深度版增加竞争对手对比+演变分析。识别创新点/差异化，评估可持续性和风险。 | 9宫格表格+可视化画布、商业模式对比图、风险评估表 |
| **3) 资源能力** | 全面版资源分析：专利、产能、认证、人力资源、研发投入、供应链、渠道、品牌。定量统计+定性评估结合，生成综合评分。支持竞争对手对比和资源缺口识别。 | 资源清单表格、核心能力雷达图、资源对比图、能力短板分析表 |
| **4) 财务分析** | 读取Excel导入的财务数据，自动计算财务比率（ROE, 毛利率，负债率等），生成趋势图。支持同业对比和异常波动预警。 | 财务趋势图、同业对比表、异常波动预警 |
| **5) 前景分析** | 结合上传的研报、管线进度文档、扩产计划，AI分析预测未来增长点。支持情景分析（乐观/中性/悲观）。 | 增长驱动力报告、管线进度条、风险-机会矩阵 |
| **6) 发展历程** | 从上传的文档中按时间轴提取里程碑事件（融资、并购、获批），生成可视化时间轴。支持发展阶段划分和关键转折点识别。 | 企业发展时间轴、发展阶段说明、关键事件列表 |
| **7) 股权/子公司** | 解析上传的股权结构图、组织架构文档，识别核心利润贡献子公司。支持股权穿透分析和关联交易识别。 | 股权穿透图、核心子公司列表、利润贡献分析表 |
| **8) 企业文化** | 分析上传的官网介绍、员工评价、社会责任报告，提取文化关键词。生成ESG评分和文化特征描述。 | 文化关键词云、ESG评分卡片、文化特征描述 |

### 5 技术选型

| 技术领域 | 选型方案 | 说明 |
| :--- | :--- | :--- |
| **前端界面** | Streamlit | Python原生，快速开发数据应用界面 |
| **文档解析** | PyPDF2 / python-docx | 解析PDF、Word文档内容 |
| **AI大模型** | Ollama本地部署 + 云端API | 本地：Qwen2.5-7B、Qwen3-8B；云端：通义千问/DeepSeek作为备选 |
| **向量数据库** | Chroma / FAISS | 存储文档向量，支持RAG检索 |
| **数据存储** | SQLite / JSON文件 | 轻量级存储，无需额外数据库服务器 |
| **可视化** | Plotly / Matplotlib | 生成图表和可视化报告 |

**AI模型选择策略：**
- **本地模型（Ollama）：** 用于常规分析任务，保护数据隐私，无API费用
- **云端模型（通义千问/DeepSeek）：** 用于复杂推理或本地模型效果不佳时

### 附录A：八大维度详细设计

#### A.1 战略分析维度

**功能定位：** 从企业年报、公告中提取战略信息，进行多维度、多层次分析

**分析模式：**
| 模式 | 适用场景 | 输出内容 |
| :--- | :--- | :--- |
| **浅层分析** | 快速了解、初步筛选 | 战略目标清单、关键举措列表、战略关键词 |
| **中层分析** | 常规研究、定期跟踪 | SWOT矩阵、战略意图解读、核心竞争力识别 |
| **深层分析** | 深度研究、投资决策 | 战略地图、竞争定位图、风险热力图、战略演变趋势 |

**对比分析类型：**
1. **横向对比：** 2家及以上企业同期战略对比
   - 战略定位差异
   - 资源配置对比
   - 竞争策略差异
   
2. **纵向对比：** 单企业3-5年战略演变
   - 战略调整轨迹
   - 关键转折点识别
   - 战略一致性评估

#### A.2 商业模式维度

**功能定位：** 分析企业商业模式构成，识别创新点和风险

**分析模式：**
| 模式 | 适用场景 | 输出内容 |
| :--- | :--- | :--- |
| **基础版** | 快速了解商业模式全貌 | 9宫格画布填充 |
| **进阶版** | 深入研究商业模式设计 | 9宫格 + 商业模式类型识别 + 盈利模式分析 |
| **深度版** | 投资决策、竞争对比 | 进阶版内容 + 竞争对手对比 + 演变分析 |

**特殊分析要素：**
1. **创新点识别：** 差异化优势、独特机制、竞争壁垒
2. **可持续性评估：** 护城河深度、抗风险能力、增长潜力（五星评分）
3. **风险识别：** 模式脆弱性、外部威胁、可复制性风险

**输出格式：** 表格形式 + 可视化画布结合

#### A.3 资源能力维度

**功能定位：** 全面盘点企业资源禀赋，评估核心能力，识别竞争优势和短板

**资源类型（全面版）：**
| 资源类别 | 具体指标 | 数据来源 |
| :--- | :--- | :--- |
| **技术资源** | 专利数量、专利质量、研发投入、技术领先性 | 专利数据库、年报研发章节 |
| **生产资源** | 产能规模、产能利用率、生产基地分布、设备先进性 | 产能公告、环评报告、官网 |
| **人力资源** | 员工数量、学历结构、核心团队、招聘趋势 | 年报、招聘JD、LinkedIn |
| **资质认证** | GMP认证、FDA认证、欧盟认证、ISO认证 | 认证公告、官网 |
| **供应链资源** | 供应商稳定性、原材料控制力、成本优势 | 供应商公告、成本分析 |
| **渠道资源** | 客户覆盖、渠道深度、大客户依赖度 | 客户公告、销售数据 |
| **品牌资源** | 品牌知名度、行业地位、ESG评级 | 行业排名、社会责任报告 |

**评估方式（定量+定性）：**
| 评估维度 | 定量指标 | 定性评估 |
| :--- | :--- | :--- |
| 规模 | 数量、金额、占比 | 行业排名、市场份额 |
| 质量 | 成功率、通过率、评级 | 技术领先性、稀缺性 |
| 增长 | 增长率、投入增速 | 发展潜力、战略重要性 |
| 风险 | 集中度、依赖度 | 可持续性、替代难度 |

**可视化组合：**
1. **资源清单表格：** 详细展示各类资源数据
2. **核心能力雷达图：** 多维度能力对比（技术、生产、人力、渠道等8个维度）
3. **资源对比图：** 与竞争对手的资源对比
4. **能力短板分析表：** 识别资源缺口和能力薄弱环节

**特殊功能：**
- **竞争对手对比：** 横向对比同类企业的资源禀赋差异
- **资源缺口识别：** 基于战略目标，识别关键资源短板
- **能力矩阵：** 评估资源-能力-竞争优势的转化关系

#### A.4 财务分析维度

**功能定位：** 自动化财务数据分析，识别财务健康度和风险信号

**分析内容：**
| 分析模块 | 具体指标 | 输出形式 |
| :--- | :--- | :--- |
| **盈利能力** | ROE、ROA、毛利率、净利率、EBITDA率 | 趋势图、同业对比图 |
| **偿债能力** | 资产负债率、流动比率、速动比率、利息保障倍数 | 预警表、趋势图 |
| **运营效率** | 存货周转率、应收账款周转率、总资产周转率 | 对比表、趋势图 |
| **成长能力** | 营收增长率、净利润增长率、总资产增长率 | 趋势图、预测模型 |
| **现金流** | 经营现金流、自由现金流、现金流覆盖率 | 趋势图、健康度评分 |

**特殊功能：**
- **异常波动预警：** 识别财务数据的异常变化（如毛利率骤降、负债率飙升）
- **同业对比：** 与竞争对手的财务指标横向对比
- **杜邦分析：** ROE分解分析（盈利能力×运营效率×杠杆）

#### A.5 前景分析维度

**功能定位：** 基于现有信息预测企业未来发展，识别增长驱动力和风险

**分析内容：**
| 分析模块 | 具体内容 | 输出形式 |
| :--- | :--- | :--- |
| **增长驱动力** | 新产品管线、新市场拓展、产能扩张、并购整合 | 驱动力清单、贡献度评估 |
| **管线进度** | 在研产品、临床阶段、预计上市时间、峰值销售预测 | 甘特图、进度表 |
| **风险因素** | 政策风险、市场风险、技术风险、竞争风险 | 风险矩阵、应对建议 |
| **增长预测** | 未来3-5年营收/利润预测（乐观/中性/悲观三种情景） | 预测表、情景分析 |

**特殊功能：**
- **情景分析：** 提供乐观、中性、悲观三种情景预测
- **敏感性分析：** 识别对业绩影响最大的关键变量
- **里程碑跟踪：** 跟踪关键事件（产品获批、产能投产）对前景的影响

#### A.6 发展历程维度

**功能定位：** 梳理企业发展历史，识别关键转折点和成长轨迹

**分析内容：**
| 事件类型 | 具体内容 | 输出形式 |
| :--- | :--- | :--- |
| **融资事件** | IPO、定增、发债、股权融资 | 时间轴、融资规模表 |
| **并购事件** | 收购、合并、资产剥离 | 时间轴、交易详情表 |
| **产品里程碑** | 新产品获批、重磅产品上市 | 时间轴、产品贡献表 |
| **产能扩张** | 新基地建设、产能投产 | 时间轴、产能增长图 |
| **战略合作** | 重要合作协议、战略联盟 | 时间轴、合作详情 |

**发展阶段划分：**
- **创业期：** 公司成立、初期产品、首次融资
- **成长期：** 快速扩张、产品线丰富、市场份额提升
- **成熟期：** 行业龙头、稳定增长、全球化布局
- **转型期：** 战略调整、业务重组、新赛道布局

**输出形式：**
- **可视化时间轴：** 交互式时间轴，支持缩放、筛选
- **发展阶段说明：** 各阶段特征、关键成功因素
- **关键事件列表：** 详细的事件描述和影响分析

#### A.7 股权/子公司维度

**功能定位：** 解析企业股权结构和组织架构，识别核心利润来源

**分析内容：**
| 分析模块 | 具体内容 | 输出形式 |
| :--- | :--- | :--- |
| **股权结构** | 实际控制人、大股东、股权比例、股权质押情况 | 股权穿透图、股东列表 |
| **子公司梳理** | 控股子公司、参股公司、联营企业、合营企业 | 子公司列表、持股比例表 |
| **利润贡献** | 各子公司营收、利润、利润贡献度 | 贡献度排名表、利润结构图 |
| **关联交易** | 母子公司交易、关联方往来、担保情况 | 关联交易表、风险提示 |

**特殊功能：**
- **股权穿透：** 识别最终实际控制人
- **核心资产识别：** 识别对企业最重要的子公司/资产
- **风险预警：** 识别股权质押风险、关联交易风险

#### A.8 企业文化维度

**功能定位：** 分析企业文化特征，评估ESG表现

**分析内容：**
| 分析模块 | 具体内容 | 输出形式 |
| :--- | :--- | :--- |
| **核心价值观** | 使命、愿景、价值观关键词 | 关键词云、价值观描述 |
| **文化特征** | 创新文化、执行文化、客户导向、团队协作 | 文化画像、特征描述 |
| **ESG评估** | 环境(E)、社会(S)、治理(G)三方面评分 | ESG评分卡片、详细评估 |
| **员工评价** | 工作氛围、薪酬福利、职业发展（如有数据） | 评价摘要、改进建议 |

**ESG评估维度：**
- **环境(E)：** 环保投入、碳排放、绿色生产、废物处理
- **社会(S)：** 员工权益、社区贡献、产品责任、供应链责任
- **治理(G)：** 董事会结构、信息披露、反腐败、股东权益

**输出形式：**
- **文化关键词云：** 高频词汇可视化
- **ESG评分卡片：** 三星/四星/五星评级
- **文化特征描述：** 企业文化画像
